# Environment

This section lists all the dependencies and environment required for **Kya Saabzi Frontend** [:fontawesome-brands-github:](https://github.com/manav-gkmit/kya-saabzi-frontend/)

---

## Installation Guide

Follow these steps to set up and run the frontend locally.

### Clone the Repository

```console
git clone https://github.com/manav-gkmit/kya-saabzi-frontend.git
cd kya-saabzi-frontend
```

### Install Dependencies

```console
npm install
```

### Start the Development Server

```console
npm run dev
```

Then you should see output like:

```console
VITE v7.2.2  ready in 500ms
Local: http://localhost:5173/
```

To connect with the backend make sure you have your `.env` file as:

```console
VITE_API_BASE_URL=http://localhost:8000
```

And poof... you should now be able to use the app locally!
